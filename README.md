# 呉市 高等学校・高等専門学校データダッシュボード

## 概要
呉市内における高等学校・高等専門学校の男女別生徒数、教員数、学校数データを可視化するダッシュボードアプリケーションです。
HTMLとJavaScriptのみで構築されており、サーバーやフレームワークは不要です。

## 機能

### データアップロード
- JSONファイルをブラウザから直接アップロード
- ファイル読み込み後、自動的にダッシュボードを表示

### フィルター機能
- 年度別フィルタリング
- 設置種別フィルタリング（国立/県立/市立/私立）
- 学校種別フィルタリング（高等学校/工業高等専門学校）

### サマリーカード
- 学校数
- 教員数
- 生徒数（計）
- 男女比

### グラフ表示
1. **性別生徒数** - ドーナツチャート（男女比の可視化）
2. **学校種別別生徒数** - 棒グラフ
3. **設置種別別生徒数** - 円グラフ
4. **年度別生徒数推移** - 折れ線グラフ（男女別の推移を表示）
5. **設置種別・学校種別別 教員数** - 横棒グラフ

### 詳細テーブル
- 全データを表形式で表示
- 検索機能
- ページネーション
- 表示件数変更（10/25/50件）

## JSONフォーマット

```json
[
  {
    "year": 2020,
    "school": 1,
    "type": "国立",
    "category": "工業高等専門学校",
    "population": {
      "teacher": 66,
      "sutudent": {
        "data": [
          {
            "type": "男",
            "population": 674
          },
          {
            "type": "女",
            "population": 206
          }
        ]
      }
    }
  }
]
```

### データフィールド説明
- **year**: 年度（西暦）
- **school**: 学校数
- **type**: 設置種別（国立/県立/市立/私立）
- **category**: 学校種別（高等学校/工業高等専門学校）
- **teacher**: 教員数（本務者）
- **sutudent**: 生徒数データコンテナ
  - **data[].type**: 性別（男/女）
  - **data[].population**: 生徒数

## 使用方法

1. `index.html` をブラウザで開く
2. 「JSONファイルを選択」ボタンをクリック
3. 対応形式のJSONファイルを選択
4. 自動的にダッシュボードが表示されます
5. フィルターで年度・設置種別・学校種別を絞り込み可能

## 特徴

- **リアルタイム集計**: フィルター変更時に自動的に統計情報を再計算
- **年次推移の可視化**: 折れ線グラフで年度別の生徒数増減を一目で把握
- **目に優しいデザイン**: パステルカラーベースの配色
- **レスポンシブ対応**: モバイルデバイスでも利用可能
